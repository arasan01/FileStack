# FileStack

Cut &amp; Paste for CLI

## Installation

All user install -> `Install.sh`
Just user install -> `NoRootInstall.sh`

`NoRootInstall.sh` is Bash Only
if you install zsh and other shell, rewrite `NoRootInstall.sh`

```
cat << EOS >> ~/.{{ your shell rc }}
alias fs='~/.fs/fs'
EOS
```

## Usage

run `$ fs`
